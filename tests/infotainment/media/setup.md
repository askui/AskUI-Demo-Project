# Media Test Setup

1. Use the `sil_mock_service` tool to call `GET media` to verify mock USB media library is loaded (should contain 12 tracks)
2. Use the `sil_mock_service` tool to call `GET radio` to verify mock FM/AM stations are available
3. From the Home screen, tap the "Audio" icon to open the Audio/Media screen
4. Wait for the Audio/Media screen to fully load
5. Verify the Audio/Media screen is displayed with source options (FM, AM, USB, Bluetooth Audio)
